# WSpace (Task Manager)

## Installion
```
pip install wspace
```

## Usage

### Initialize Database

```bash
wspace init
```

### Add Tasks
```bash
wspace add "Make Everything done"
```

### List All Tasks (Start from 0)
```
wspace tasks

+----+----------------------+
| ID |        Tasks         |
+----+----------------------+
| 0  | Make Everything done |
+----+----------------------+
```

### Delete tasks
```
wspace del 0
```
